# N26csv2qif
Convert N26 CSV to QIF (based on Rabobank csv2qif)

## Requirements
Python

N26 CSV file
  
# Instuction
 
Place CSV file in same folder as n26csv2qif.py and run the following command:
```
pyhton n26csv2qif_v1.py inputfile.csv
```
